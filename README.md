# 🎬 Movie Review Analytics System

AI-powered Movie Review Dashboard built using NLP, Machine Learning, Streamlit, SQLite, and Plotly.

## Features

- AI sentiment prediction
- Review analytics
- Glassmorphism UI
- Movie posters
- Interactive charts
- Top-rated movies
- Trending movies
- Review filtering
- Database storage
- Timestamps

## Technologies Used

- Python
- Streamlit
- Scikit-learn
- NLTK
- SQLite
- Plotly
- Pandas
